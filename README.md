**TDL** Tidal Downloader Installer

## Supported OS
- MacOS

## Requirements
- python3
- ffmpeg
- python distribution
- tidal-dl

## Installation
1. Open the Terminal application and type the following commands


```bash
2. 'git clone https://github.com/bouzy/tdl.git'
3. 'cd tdl'
3. 'sh setup.sh'
```
